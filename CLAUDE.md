# Claude Code Instructions

Read and follow [AGENTS.md](./AGENTS.md) for all project-specific instructions.

## Additional Claude-Specific Rules

### Commit Style
- Use conventional commits: `feat:`, `fix:`, `docs:`, `chore:`, etc.
- NEVER add "Co-Authored-By", "Generated by", or any AI attribution
- Keep commit messages concise but descriptive

### When Making Changes
1. Always run `pnpm test` before suggesting commits
2. Verify builds work with `pnpm build`
3. Check for TypeScript errors with `pnpm typecheck`

### Memory (Engram)
This project uses Engram for persistent memory. Key topic keys:
- `sdd-init/react-native-config-ultimate` - Project context
- Search with: `mem_search(query: "...", project: "react-native-config-ultimate")`

### Skills to Load
When working on this project, relevant skills include:
- `react-native-best-practices` - For RN-specific patterns
- `typescript` - For strict TypeScript patterns
